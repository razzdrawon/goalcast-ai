# GoalCast AI

AI-powered World Cup 2026 prediction platform. Predict every match, compete on leaderboards, and win prizes at official watch party venues.

## What is GoalCast AI?

GoalCast covers all 104 World Cup 2026 matches. Fans predict results, earn points, and compete locally at partnered restaurants and bars. AI analyzes team form, historical data, and match context to help fans make smarter predictions.

## For Fans
- Predict every match result and exact score
- Compete on the Atlanta leaderboard
- Play live trivia during watch parties
- Win prizes sponsored by local venues

## For Venues
- Get listed as an Official Watch Party location
- Receive weekly fan attendance reports before each match
- Drive foot traffic and engagement during the tournament

## Tech Stack
- **Backend:** NestJS + TypeScript
- **AI Service:** Python + FastAPI + LangGraph
- **Frontend:** React + TypeScript
- **Database:** PostgreSQL + Redis
- **Messaging:** Apache Kafka
- **Deployment:** AWS ECS + Docker
